## Overview

Thermal Engine v8.0 is a custom-tuned thermal configuration designed for  
**Snapdragon¬Æ 7s Gen 2** devices running **AOSP-based ROMs**.

The goal is to deliver **balanced performance**, **better thermal control**,  
and **optimized charging behavior** without unsafe or extreme limits.

---

## Features

- Improved overall temperature management
- Optimized charging behavior
- PUBG-specific thermal configuration
- Reduced aggressive throttling
- Custom performance-boost thermal profiles
- Tuned for daily usage and gaming stability
